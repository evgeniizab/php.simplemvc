# simple MVC

### Requirements
   The requirements to application is:
   *    **Docker**
   *    **Composer**

### Installation

#### 1. Git Clone
   ```sh
   $ git clone https://github.com/evgeniizab/php.simplemvc.git
   $ cd php.simplemvc
   $ cp .env.docker .env
   $ composer install
   $ docker-compose up -d --build
   ```
#### 2. Open the application in a browser
